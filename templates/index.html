What is this?

AltAI is a curated directory of alternatives to the most popular AI tools. Every alternative is picked for a specific reason — "cheaper", "open-source", "better long-form writing", "runs locally" — not because someone paid to be listed.

The directory is fully static. No tracking, no cookies, no login. Data comes from a single JSON file that anyone can audit.

We earn a commission if you sign up for a tool through one of our affiliate links. That's it. It never influences which tools appear or how they're ranked.
