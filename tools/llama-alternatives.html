Llama Reference

Llama is Meta's family of open-weight language models — from Llama 3.1 8B to Llama 3.3 70B. Free to download, fine-tune, and deploy without API costs. Powers hundreds of apps, local AI tools, and enterprise deployments. Llama 3.1 405B is competitive with GPT-4 on many benchmarks.

Strengths

  • fully open-source
  • run locally (no data sent to Meta)
  • customisable and fine-tuneable
  • no per-token costs at self-hosted scale

Weaknesses

  • requires technical setup to self-host
  • slower than API services without GPU
  • no built-in web search
