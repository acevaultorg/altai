n8n Reference

n8n is a fair-code workflow automation tool you can self-host for free. 400+ integrations, visual node editor, and full code access via JavaScript/Python. AI-native with LLM nodes built in.

Strengths

  • self-hostable (free)
  • open-source
  • code nodes for custom logic
  • AI/LLM built-in
  • no per-execution pricing

Weaknesses

  • setup complexity vs Zapier
  • smaller integration library
  • community support only on free
