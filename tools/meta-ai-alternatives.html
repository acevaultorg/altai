Meta AI Reference

Meta AI is a free assistant built on Llama 4, Meta's latest open-source model family. Available in WhatsApp, Instagram, Facebook, and at meta.ai. Llama 4 Scout and Maverick bring a 10M token context window and competitive performance against GPT-4o.

Strengths

  • completely free
  • WhatsApp and Instagram integration
  • Llama 4 is genuinely competitive
  • open-source Llama for self-hosting

Weaknesses

  • data used to improve Meta's AI
  • no code interpreter or image generation
  • less polished than ChatGPT
