Microsoft Copilot Reference

Microsoft Copilot is the AI layer across Microsoft's product ecosystem — built into Windows 11, Edge browser, Bing search, and Microsoft 365 apps. It runs GPT-4o under the hood and gives you a capable AI assistant at no extra cost if you already use Microsoft products. Copilot Pro unlocks deeper Office integration and faster responses.

Strengths

  • Free for most users — built into Windows and Edge
  • Real-time Bing web search in every answer
  • Integrated with Office 365 (Word, Excel, Outlook)
  • Free DALL-E 3 image generation
  • No account required for basic use

Weaknesses

  • Less capable than Claude on complex reasoning
  • No persistent memory across sessions by default
  • Shares data with Microsoft
  • Limited compared to ChatGPT Plus on tool use
