Aider Reference

Aider is an open-source command-line AI coding assistant. It connects to Claude, GPT-4, or local models and can edit multiple files in a repo simultaneously based on chat instructions.

Strengths

  • 100% open-source and free
  • multi-file edits from terminal
  • works with any model (Claude, GPT-4, local)
  • Git integration built-in

Weaknesses

  • CLI-only, no GUI
  • requires technical setup
  • you pay for model API usage separately
